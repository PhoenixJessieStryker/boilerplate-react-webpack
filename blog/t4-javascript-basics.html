Explain the difference between accessing data from arrays and objects

An array Really just allows you to keep a Multitude of Data pieces in a single variable. They are each assigned their own numerical value which starts at zero and can be used to recall them. Objects can hold multiple values as well, but their data is defined in pairs of a property name with a corresponding value and you access their data by their property name.
